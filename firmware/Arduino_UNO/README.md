# Arduino UNO Firmware

This firmware controls the physical traffic junction.

## Functions

- Traffic Light Control
- Servo Motor Control
- IR Sensor Detection
- UART Communication with ESP32

## Hardware

- Arduino UNO
- 2 × SG90 Servo Motors
- 2 × IR Sensors
- LEDs

## Board

Arduino UNO R3

## Language

Arduino C++

## Upload Speed

115200 Baud
